===== Produkte abrufen

Eine einfache `Get` Anfrage auf ``/api/products`` gibt eine Liste aller relevanten Informationen aller Produkte zurück. +
Ein Produkt hat folgende Attribute

* id: int
* name: Char
* beschreibung: String
* preis: int
